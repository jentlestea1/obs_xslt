The OBS Framework Project

The OBS Framework Project is an R&D project of P&P Software GmbH. Its objective is to develop an object-oriented software framework for On-Board Software (OBS) and to test the applicability to it of generative programming and aspect oriented programming techniques. This project is partially supported by the European Space Agency under Estec contract 17475/03/NL/LvH/nv. The project manager and chief designer of the OBS Framework is Alessandro Pasetti.

The OBS Framework Project is currently under way. The information reported in this web site is therefore preliminary and subject to continuous update. This web site only gives access to the public part of the OBS Framework. In particular, links to source code (both C++ and XSL) are not active and attempt to download the corresponding files will results in an error. Full access to the OBS Framework can be obtained by contacting P&P Software.
