The OBS Framework Project

The OBS Framework is a research prototype of an object-oriented software framework for satellite on-board applications. Its purpose is to investigate and demonstrate how object-oriented and framework technology can be used to develop reusable and adaptable components for on-board applications.

The OBS Framework is currently maintained at the Automatic Control Laboratory of ETH-Zurich by Alessandro Pasetti and Vaclav Cechticky. The OBS Framework project has its roots in the AOCS Framework Project. The bulk of its development was done by P&P Software (a research spin-off of ETH-Zurich).

All results of the OBS Framework Project, including all its source code, are available as free and open software under GNU General Public Licence.
